Usage Instructions for the Avatar Creator Demo


Brief Description of Functionality:

  • The IMI avatar creator demo was designed to quickly generate xml configuration files for avatars.  Future changes and upgrades will better make use of the tool as a device for artists to test their assets before submission.  The tool allows for the following:
    • Load random configurations of avatars
    • Modify the avatar (joint manipulation, mesh swaping, texture swaping).
    • Saving configurations
    • Loading previously saved configurations
    • View how the differnt meshes fit with the other meshes (ie determine if a head fits onto a shirt w/o tearing)
    • Manipulate the individual nodes that comprise of the avatar


Loading An Avatar:

  • The viewer is designed to randomly create an avatar configuration based on the gender specified
  • Loading instructions:
    • Press either the random male avatar button (second button from the top right bar) or the random female avatar button (right most button on the top bar)
  • Known Issues:
    • When you randomly create an avatar it loads the random selection of files from our webserver.  This can be a bit slow. 
    • Unfortunately there is an issue with the random creation and the progress bar (ie the progress bar doesn't update)... It does however work for all the other loadings.               


Swapping Meshes:

  • The meshes that the user can swap out / in on the avatar are seperated into the following catagories:
    • Head - The avatar head
    • UpperBody - The torso and arms of the avatar (this means shirts and stuff)
    • LowerBody - The waist and legs of the avatar (this means pants and shorts stuff)
    • Feet - The ankle/feet of the avatar (this means shoes and sandles)
    • Hair - The head hair of the avatar (basically just hair)
    • Jacket - Adds onto the UpperBody and doesn't remove what is already on the UpperBody (this means jackets and coats)
    • Glasses - Adds onto the the face of the avatar
    • Hat - Adds ontop of the head of the avatar
  • Loading instructions:
    • File -> Load Model
    • Choose what catagory of model you want to load:
      • Avatar - select a folder which contains all the assets which make up an avatar **DO NOT USE THIS FOR NOW**
      • Head - select the collada (*.dae) or binary file (*.bhf) of a head to load onto an avatar. 
        • selecting this option will ask the user to select if the head is normal mapped or not.  This selection determines the correct shader the system will use.
      • Clothes - select the collada (*.dae) file of the clothes you want to load. (includes shirts, pants, shoes, & jackets)
        • selecting this option will ask the user to select which subgroup the article of clothing will go into:
          • UpperBody
          • LowerBody
          • Feet
          • Jacket
      • Accesories - select the collada (*.dae) file of the accessory you want to load. (includes hats, glasses, & hair)
        • selecting this option will ask the user for the subgroup the accessory will go into:
          • Hair
          • Hat
          • Glasses
        • after subgroup selection, the user must select the joint to add it onto
          • Head
          • Neck
        • after selecting the joint, the user must input the name of the geometry to add (the name of the mesh in the file ie mt_spikeyHair)
    • When the progress bar stops (is gray) and Loading... is no longer visible in the status bar, the screen will update with the changes.
  • Known Issues:
    • There is no undo so be sure of what you are doing...


Swapping Textures & Shaders:

  • The user can change the textures on the meshes as well as the shader being used on that piece of geometry.  The tool allows the user to create a new texture for a selected mesh or replace the existing texture with one that has already been created.  The user can also create a shader to be used on that mesh or load an existing shader template and manipulate the values.
  • Loading instructions:
    • Tools -> Texture Editor

Changeing Colors:

  • The user can change the color mask on the meshes.  The tool allows the user to change the color mask on the mesh based on the the type (ie shirt, pants, shoes, skinetone, etc...).  The user toggles the button for the category (more than one can be toggled if they user wants them to be the same color), and then color change is done by selecting color swatches or moving the marker on the color gradient (HUE, SATURATION, etc).
  • Loading instructions:
    • Tools -> Color Selector
    • Toggle the category of meshes to affect
    • Choose the color to apply.  As soon as the color is selected, the user can see the effect on the avatar in the window.
  • Known Issues:
    • Sometimes there are visual artifacts when the color changes, but when the color change is complete it is back to normal.
